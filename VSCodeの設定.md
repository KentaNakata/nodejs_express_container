# VSCodeの設定

- VSCodeのSettingを開き、右上の小さなアイコンから`setting.json`を開き、設定を追加する。必要に応じて拡張機能をインストールする。
```
    "[javascript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
    },
    "[json]": {
        "editor.defaultFormatter": "vscode.json-language-features"
    },
    "[jsonc]": {
        "editor.defaultFormatter": "vscode.json-language-features"
    }
```
